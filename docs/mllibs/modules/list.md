# Available Modules

Any type of module that works with data can be added to mllibs, however itself contains modules aimed related to data science and analysis topics. The details of each module is available in the **Documentation** section

<br>

<div class="grid cards" markdown>

- [:fontawesome-brands-html5: __Statistical Tests__ [mstats_tests]](../documentation/mstats_tests.md)

    ---

    The purpose of this module is to provide the user with the ability to do **data sample comparison tests** that are available in **statsmodels** and **scipy.stats** libraries. The module requires **list or parameter value** for sample comparison to test statistical hypotheses and is not aimed at dataframe based data
