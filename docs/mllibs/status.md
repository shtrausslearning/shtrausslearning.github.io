### :fontawesome-regular-comment-dots: User Recommendation System

<h4>Description</h4>

Currently there doesn't exist a mechanism that would allow a user to explore the functionality of the library content, nor are there mechanisms that would guide a user during the process doing a project. There is a need to develop an accurate and effective user recommendation system that that would provide personal recommendations based on the user requests, as well as general trends/interactions 

<h4>Purpose</h4>

The purpose of such functionality is to provide helpful suggestions, tips based on information the user provides, data they load & the functions they utilise
