    User Recommendation System¶

    Description

    Currently there doesn't exist a mechanism that would allow a user to explore the functionality of the library content, nor are there mechanisms that would guide a user during the process doing a project. There is a need to develop an accurate and effective user recommendation system that that would provide personal recommendations based on the user requests, as well as general trends/interactions

    Purpose

    The purpose of such functionality is to provide helpful suggestions, tips based on their preferences

    Approval

    In motion

    Module Function Label Classifier¶

    Description

    In order for module functionality to be activated, there needs to be an inbuilt library functionality that would link an input user request to module functionality

    Purpose

    The purpose of such as classifier is to associated input text requests with module functions

    Approval

    not available
